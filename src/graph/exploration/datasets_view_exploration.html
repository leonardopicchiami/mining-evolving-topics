Dataset Exploration Analysis¶

This file shows the structure of each graph for each year of the ds-1 and ds-2 datasets. Each graph, for both datasets, is constituted by several connected components; obviously being ds-2 enormously bigger compared to ds-1, the connected components are in greater number for every year of the timeline.

    timeline.append(str(i))

Creation, for each year of the timeline, of the ds-1 and ds-2 dataset graphs and addition of the YearGraph object of each given year, containing the two graphs for ds-1 and ds-2, to a dictionary where the key is the year and the value is the object itself.

In [3]:
timeline_graphs = dict()

for year in timeline:
    graph = gr.YearGraph(ds1_path, ds2_path, year)
    graph.read()
    graph.build_ds1_graph()
    graph.build_ds2_graph()
    timeline_graphs[year] = graph

In the next pieces of code the ds-1 ds-2 graphs will be plotted for each year so that we can analyze the structure
